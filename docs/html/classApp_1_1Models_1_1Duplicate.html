static App\Models\Duplicate::compute ( Crew  $crew)
static

This function is used to get similar persons. It compares the similarity of the persons based on the fields that are completed. It computes the similarity of each field and then averages the similarity of each field to get the similarity of the persons for a given team. It must not compute the similarity of a person with itself. It must not compute the similarity of a person with a person that has already been compared. (eg A -> B is the same as B -> A) It must not compute the similarity of a couple that has already been compared and not edited since. It must not compute the similarity of a couple marked as "not similar" (ie resolve) by the user.
